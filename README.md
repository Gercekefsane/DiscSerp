Web PANEL ScreenShoot

- Sending Server Member + Emoji ( Can bypass Server Rules)

- Sending Messages to Channel
- Sending Dm Spam
- Sending Friend Sender
- Sending Server Boost (Like x2 x4 x6 x8 x10 x12 x14 and more!!)
